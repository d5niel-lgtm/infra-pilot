# Infra Pilot

**Orchestrate. Automate. Scale.**

Infra Pilot is a modular orchestration framework for managing containers, cloud resources, and game servers through a unified CLI, web dashboard, or Discord bot — across AWS, Azure, GCP, Hetzner, or local infrastructure.

## Quick Start

```bash
git clone https://github.com/DaaanielTV/infra-pilot.git
cd infra-pilot
cp .env.example .env
docker compose up -d
```

| Service | URL |
|---------|-----|
| Management Panel | http://localhost:5173 |
| API | http://localhost:3001 |
| Orchestrator Health | http://localhost:8500/health |

## Key Features

- **Multi-Cloud Orchestration** — Unified API for AWS, Azure, GCP, Hetzner, DigitalOcean, and more
- **200+ CLI Commands** — Full infrastructure control from the command line
- **Discord Integration** — Provision and manage servers directly from Discord
- **Green Computing** — Track energy usage, offset CO2, and rank providers by sustainability
